# whatsapp-helper-bot

## Getting Started

### 1) Create a `.env` file
Create a file named `.env` in the project root and add your webhook URL:

```env
WEBHOOK_URL=http://localhost:<port>/webhook/<your-webhook-path>
```

Example:

```env
WEBHOOK_URL=http://localhost:5678/webhook/whatsapp-helper-bot
```

### 2) Install dependencies

```bash
npm install
```

### 3) Start the bot

```bash
npm run start
```
